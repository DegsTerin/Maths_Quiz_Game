# v1.0.0

## Highlights

- Published the first public version of the Maths Quiz Game project.
- Reorganised the repository for GitHub with clear folders for Arduino code, hardware files, design assets, documentation, media, and simulation files.
- Converted the Arduino sketch, comments, and documentation to English.
- Added project photos, a Wokwi simulation setup, and a cleaner README.

## Included in this release

- Main Arduino sketch in `arduino/maths-quiz-game/maths-quiz-game.ino`
- Local simulation files in `simulation/`
- Electronics and design source files in `hardware/` and `design/`
- Project documentation and media assets for presentation

## Notes

- The repository currently tracks `assets/media/demonstration.mp4` through Git LFS.
- If inline playback on GitHub is required, replace the LFS-tracked pointer with a normal compressed video file below GitHub's standard file size recommendation.
